Chapter 4. 반복문#

학습목표

반복(iteration)문은 동일한 문장을 반복해서 실행하는 구조이다. 이 장에서는 반복문 while과 for에 대해서 알아본다.

While loop#

먼저, while문에 대해 알아보자.

while statement#

while문의 형식은 다음과 같다. 조건 뒤에는 반드시 콜론(:)이 있어야 하며, 조건이 True인 동안 while문 안에 있는 실행 문장을 실행한다. 이때 실행 문장은 반드시 들여쓰기가 되어 있어야 한다.

while 조건:
    실행 문장1
    실행 문장2
    ...

  • 예를 통해서 while문을 이해해보자.

  • 아래 코드에서는 반복을 위한 조건으로 변수 count를 활용했다. 변수를 조건에 활용하기 위해서는 반드시 while문 이전에 어디에선가 미리 정의되어 있어야 한다. while문 이전에 변수 count를 1로 정의하였다.

  • 변수 count는 while문 안에서 반복할 때마다 1씩 증가한다.

  • 그러다가 count가 5보다 크면, 즉 6이 되면, 조건 (count <= 5)이 False가 되어 반복을 멈춘다.

count = 1
while count <= 5:
    print(f'재미있는 파이썬:{count}')
    count=count+1
        
print(f'count={count}')

재미있는 파이썬:2
재미있는 파이썬:3
재미있는 파이썬:4
재미있는 파이썬:5
count=6

break#

  • while문의 조건 부분에 조건식이 아닌 True 또는 1 등을 넣어줄 수도 있다. 이 경우에는 while True: 또는 while 1: 이므로 조건이 항상 True가 되어 무한 반복하게 된다. 따라서 반드시 반복문에서 탈출하는 break를 반복문 안 어디엔가 써줘야 한다.

count = 0
while True :
    print(f'재미있는 파이썬:{count}')
    count = count + 1
    if count == 5:
       break
print(f'count={count}')

재미있는 파이썬:0
재미있는 파이썬:1
재미있는 파이썬:2
재미있는 파이썬:3
재미있는 파이썬:4
count=5

프로그래밍 제어구조

프로그래밍에서 True는 1의 값을 가지며, False는 0의 값을 가진다. 엄밀히 말하면 False와 0을 제외한 모든 값은 True의 의미를 가진다. 따라서 while ‘a’: 와 같이 적어도 무한 반복된다.

while ~ else#

while문에도 else가 있다. while문의 else 파트는 while문의 조건식이 False가 되서 빠져나오는 경우에만 실행한다. 즉, while문이 정상적으로 모든 반복을 다 한 경우에만 실행한다. break를 통해서 while문을 빠져나온 경우에는 else문이 실행되지 않는다.

count = 0
while count < 5 :
    pw=input('password')
    if pw=='pythonisfun':
       print('로그인 되었습니다.')
       break
    count = count + 1    
else:
    print ('로그인에 실패했습니다.')

continue#

반복문에서 continue를 만나면 continue 아래쪽에 있는 모든 코드는 실행하지 않고 while문이 있는 라인으로 이동한다.

i = 0
while i < 6:
    i += 1
    if i==3 or i==4:
        continue
    print(i)

for loop#

while문이 주어진 조건에 대해 반복하는 것이였다면 for문은 주어진 범위에 대해 반복한다.

for statements#

for 변수 in 범위:
    statements
next_statements

여기서 범위에는 range, 문자열, 리스트, 튜플 등 묶음으로 반복가능한 타입들이 올 수 있다.

  • for문과 함께 유용하게 사용되는 range() 함수를 먼저 알아보자.

  • range() 함수는 일정한 규칙에 따라 나열된 수열을 생성한다.

range(start, stop, step)

  • start: 범위의 시작 값으로 생략하면 0이다.

  • stop: 범위의 끝값+1

  • step: 범위의 간격으로 생략하면 1이다.

  • 1부터 9까지 수열을 생성한다.

  • 시작 값 생략으로 0부터 9까지 수열을 생성한다.

  • 3부터 9까지 3씩 간격으로 수열을 생성한다.

range(1, 10) 
range(10)   
range(3, 10, 3) 

for count  in range(1, 11):
    print(f'나무를 {count}번 찍었습니다.')
print ('넘어가즈아~~~')

  • 반복문안에서 값을 누적시키기 위해서는 누적을 위한 변수를 반복문 밖 위쪽에 정의해야 한다.

total=0

  total = total+i   
  print(total)

print("The total is", total)

  • 범위에 문자열을 넣을 수 있다.

for i in 'Hello World':

for~else#

for문에도 else가 있다. while ~ else와 마찬가지로 for문이 정상적으로 모든 반복을 다 한 경우에만 실행한다.

a = 'computer'
letter = 'u'

for i in a:
    if i == letter:
        print(f'{letter}가 검색되었습니다.')
        break
else:
    print(f'{letter}는 검색되지 않았습니다.')
print('END')

마무리#

  • 프로그래밍 제어 구조는 순차, 선택, 반복 구조가 있다.

  • 반복구조는 조건 또는 범위에 따라 실행문장을 반복한다.

  • while문은 조건비교 결과가 True인 동안 블록 소스코드를 반복 수행한다.

  • for문은 범위의 처음부터 끝까지 차례대로 반복한다.

  • for문의 in 연산자 뒤에는 차례가 있는 숫자, 리스트, 문자열 등이 올 수 있다.

  • for문의 for 변수는 in 뒤에 있는 자료값을 하나씩 차례대로 저장한다.

  • for문은 숫자값의 범위를 만들어주는 range()함수를 유용하게 활용한다.
