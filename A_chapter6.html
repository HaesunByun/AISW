6장. 함수#

  • 프로그램에서 자주 사용되는 코드를 만들어 두고 필요할 때마다 호출해서 사용하는 기능(기능 분리, 재사용)

  • 함수의 필요성

  • 코드의 간결성 :코드가 중복되지 않고 간결해진다.

  • 코드의 재사용성:한 번 작성해둔 코드를 여러 번 사용하므로 코드를 재사용할 수 있다.

  • 코드 수정의 용이성 : 프로그램 기능을 함수로 나누어 묶기 때문에 코드 수정이 쉽다.

  • 프로그램의 모듈화 : 기능별로 함수를 작성하므로 프로그램 모듈화가 증대된다.

  • 종류

  • 내장함수 : int(), str(), print(), input()…….

  • 사용자정의함수 : 프로그래머가 필요에 의해서 만든 함수

def mySum():
  total=0
  for num in range(1,101):
      total = total + num
  return total #함수안에서 나온 결과를 함수 밖에서 사용하고자 할때에는 반드시 return이 있어야 함

변수의 유효 범위( Variable scope)#

Global variable vs. local variable

def func1():
    v1=50 #local variable

def func2():
    v2=70 #local variable

func1()  
print(v1)

def func1():
    v1=50
    print(v2)

def func2():
    v2=70
    func1() # func1() 호출

func2() # func2() 호출

v2 = 3 #global variable
def func1():
    global v2
    v1=50
    v2 = v2 + 1
    print(v2)

def func2():
    v2=70
    print(v2)
    func1()

func2()
print(v2)
